1. CREATE DATABASE "Database Name"; =======> for creating a new Database
  2. USE "Database Name"; =======> for using a exist database
  3. CREATE TABLE "Table Name"( Data Type); ========> for create a new table
  4. DROP TABLE "Table Name"; ========> FOR DELETE A TABLE.
  5. INSERT INTO "TABLE NAME" ("DATA TYPES") VALUES ("VALUE OF DATA"); ========> Insert data in a table.
  6. SHOW "DATA TYPE" FROM "TABLE NAME" ========> SHOW TABLE DATA FROM TABLE, IF YOU WANT TO SEE ALL TABLE REPLACE "DATA TYPE" BY "*"
  7. SET SQL_SAFE_UPDATES = 1 ============> FOR UPDATING ANY DATA IN DATABSE. THIS IS CALLED SAFE MODE OFF.
